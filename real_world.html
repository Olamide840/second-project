WannaCry Ransomware Attack

What Happened:

  • WannaCry spread via a vulnerability in Windows (EternalBlue).
  • It encrypted users files and demanded Bitcoin payments.

Impact:

  • Affected 200,000+ computers in 150+ countries.
  • Hospitals, transport, and corporations were disrupted.

Prevention:

  • Install security patches regularly.
  • Use antivirus software and offline backups.
  • Disable SMBv1 if unused.

Equifax Data Breach (2017)

What Happened:

  • Hackers exploited an Apache Struts vulnerability.
  • Sensitive data of 147 million Americans was exposed.

Impact:

  • Huge reputational damage and lawsuits.
  • Massive financial losses for Equifax.

Prevention:

  • Promptly apply security patches.
  • Regular vulnerability scanning and monitoring.
